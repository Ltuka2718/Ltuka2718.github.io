基础模块的导入

import requests
from bs4 import BeautifulSoup

请求并解析即将爬取的网页

xxx_url =''
headers = {
"'你在登录信息中php文件中查看到的headers信息，一般在最后一行，格式如下'"
'User-Agent': xxx
}
#获取信息并开始解析
xxx = requests.get(xxx_url, header = headers)
bs = Beautiful(xxx.text, 'html.parser')

以上的.text是进行文本化的解析，后面’html.parser’对解析器进行一个选择

如何获取Cookies消息

对页面进行审查后，我们可以在登录信息的payload区找到如下一串

log: username
pwd: password
wp-submit:""
redirect_to:""
testcokkie:""

这个会成为我们爬取网站的请求体数据。我们的用户名和密码还是在命令行中输入

请求登录的时候我们携带请求体对网站进行post请求

login = requests.post(url, headers=headers, data=data)
print(login.cookies)

发送评论

我们可以先在审查元素中找到，评论的表单
寻找请求网站和请求体表单

cookies = login.cookies
comment = requests.post(url, headers = headers,data = data_comment,cookies = cookies)
print(comment.status_code)

打印状态码是我们了解程序结果重要的一部分

这里的cookies的引用方法，也保证了我们的程序对于another也是适用的
